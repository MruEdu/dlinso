app.py가 매 대화 턴마다 시스템 지시문을 만들고, 브랜치 app_mode에 따라 모듈이 프롬프트 조각을 이어 붙입니다. On each turn, app.py builds the system instruction; modules append prompt fragments based on branch app_mode.

파일 역할 File roles

  • app.py — 세션(연령·생애주기·phase)을 읽고 build_system_instruction() 호출 Reads session (age, life stage, phase) and calls build_system_instruction()
  • modes/registry.py — lifespan / assessment / counseling / research 브랜치 메타데이터 Branch metadata for lifespan, assessment, counseling, and research modes
  • prompts/registry.py — app_mode에 맞는 프롬프트 모듈로 위임 Delegates to the prompt module matching app_mode
  • prompts/lifespan.py — 현재 활성: Phase1(회상) · Giant(거장) 조각 조립 Currently active: assembles Phase 1 (recall) and Giant phase fragments
  • personas.py — 연령·학력 말투, 거장 페르소나 문구(실제 본문) Age/education tone and giant persona copy (core prompt text)
